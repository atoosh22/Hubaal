HTML & CSS

Course Description

In this course, you will learn the fundamentals of web development using HTML and CSS. You’ll discover how to structure web pages with HTML elements, add text, images, and links, and then style them beautifully using CSS. The course will guide you through creating layouts, using colors and fonts, making pages responsive for all devices, and understanding best practices in modern web design. By the end, you’ll be able to build your own professional and visually appealing websites from scratch

Duration: 3 months

Instructor Info

Mohamed Abdulaahi is an experienced web developer and educator with a strong background in HTML, CSS, and modern web design techniques.
